# Raycast Shell Scripts

A collection of custom [Raycast Script Commands](https://github.com/raycast/script-commands) for macOS automation.

## Scripts

| Script | Description |
|--------|-------------|
| **Timestamp** | Copies current timestamp `[YYMMDD-HHMM]` to clipboard |
| **X to Twitter URL** | Converts `x.com` URLs in clipboard to `twitter.com` |

## Setup

1. Open Raycast Settings
2. Go to **Extensions** > **Script Commands**
3. Add this directory as a script command folder

## Requirements

- macOS
- [Raycast](https://raycast.com/)
